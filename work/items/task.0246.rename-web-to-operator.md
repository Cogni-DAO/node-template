## Context

The ROADMAP, node-operator-contract spec, and cicd-services-gitops project all
reference `apps/operator/` as the target name. The current `apps/web/` name is a
holdover from when this was a single-app repo.

This is a pure rename — no functional changes. But the blast radius is large
(~70 files, references in configs, CI, scripts, dep-cruiser, tsconfig, Docker).
Deserves its own focused PR to keep the diff reviewable.

## Plan

1. `mv apps/web apps/operator`
2. Update `apps/operator/package.json` name field
3. Find-and-replace `apps/web` → `apps/operator` across all config files
4. Find-and-replace `--filter web` → `--filter operator` in root package.json
5. `pnpm install` (regenerate lockfile)
6. `pnpm check` (validate everything)
7. If dep-cruiser tests fail, update the path patterns in test fixtures
